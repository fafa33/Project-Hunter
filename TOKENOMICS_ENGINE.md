# TOKENOMICS ENGINE

## Purpose

The Tokenomics Engine evaluates whether a token's economic design creates long-term value for token holders.

The objective is not understanding token mechanics.

The objective is determining whether long-term demand can exceed long-term supply.

---

# First Principle

Technology creates products.

Tokenomics creates investments.

A great protocol can produce a poor investment if value does not accrue to the token.

---

# Core Question

For every project answer:

Why should someone own this token instead of simply using the protocol?

If the token disappeared tomorrow,

would the business still function?

If yes,

the token deserves a lower score.
