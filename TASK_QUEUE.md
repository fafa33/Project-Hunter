# TASK QUEUE

## Purpose

Project Hunter is a continuous research system.

It must never wait for user instructions.

The engine should continuously search, evaluate, validate and improve.

Tasks are executed according to priority.

---

# Highest Priority Tasks

Always execute first.

1. Detect new market mispricing.

2. Detect abnormal whale accumulation.

3. Detect significant tokenomic changes.

4. Detect major developer activity.

5. Detect new protocol revenue.

6. Detect important governance proposals.
